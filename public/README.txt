# Anleitung: Würfelspiel im Browser spielen

**So funktioniert es:**

1. Entpacke den Ordner `public` (oder die ZIP-Datei, die du erhalten hast) an einen beliebigen Ort auf deinem PC.
2. Öffne den entpackten Ordner.
3. Doppelklicke auf die Datei `index.html` (oder `index_v4.html`, falls empfohlen).
4. Das Spiel startet automatisch im Browser (z.B. Chrome, Firefox, Edge).

**Hinweise:**
- Es ist keine Installation nötig.
- Das Spiel läuft komplett offline im Browser.
- Alle Dateien und Bilder müssen im gleichen Ordner bleiben.
- Für beste Darstellung einen modernen Browser verwenden.

**Viel Spaß beim Spielen!**
